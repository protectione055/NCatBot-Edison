# SystemStatus 插件

## 简介

这是一个 NcatBot 插件模板。

## 功能

- 获取系统信息
    - 系统类型
    - 系统版本
    - 处理器
    - 内存
    - 磁盘
    - Python 版本
    - 等等
- 获取状态信息
    - 处理器用量
    - 内存用量
    - 磁盘用量
    - 开机时间
    - 等等

## 使用方法

1. 使用 `/system` 获取系统信息
2. 使用 `/status` 获取状态信息

## 配置项

- `include_ip`: 是否在系统信息中展示 IP 地址，可选值为 `True` 和 `False`，默认关闭，请自行斟酌是否开启
